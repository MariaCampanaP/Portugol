# ☕︎ Portugol
Repositório contento exercícios e projetos em **Portugol** - códigos feitos para treinar lógica de programação e algoritmos usando VisualG3

## ☕︎ Sobre o Portugol
**Portugol** é uma linguagem de programação didática, escrita em português estruturado, ideal para iniciantes aprenderem lógica de programação. Seus conteúdos incluem variáveis, operadores, estruturas condicionais (como **se**, **senão**), laços de repetição (**enquanto**, **para**), funções, vetores, matrizes e manipulação de dados de forma simples e visual.

## ☕︎ Sobre o Repositório

Este repositório reúne implementações e conceitos fundamentais de **Lógica de Programação**, abordando tópicos como:

- *Estruturas Sequenciais*
- *Estruturas Condicionais*
- *Estruturas Repetitivas*
- *Vetores*
- *Matrizes*

## ☕︎ Tecnologias Utilizadas

- VisualG3
